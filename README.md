# Texas Hold'em Poker


## Prerequesites
```
pip install -r requirements.xtx
```

## Usage
Run the main program in debug mode to see all the player hands.

```
python main.py -d
```

Run the main program to see the required two cards to make the selected winner.

```
python main.py
```
